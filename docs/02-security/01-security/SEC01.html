SEC01 セキュリティ基盤

ワークロードを安全に運用するための包括的なベストプラクティス。すべてのセキュリティ領域に適用する基盤的な実践方法を提供します。

SEC01-BP01

アカウントを使用してワークロードを分離する

マルチアカウント戦略により、環境とワークロード間に共通のガードレールと分離を確立します。

高リスク

SEC01-BP02

アカウントルートユーザーとプロパティを保護する

ルートユーザーの適切な制御を確立し、日常的な使用を避けることでリスクを軽減します。

高リスク

SEC01-BP03

制御目標を特定し検証する

コンプライアンス要件と脅威モデルから制御目標を導出し、継続的に検証します。

高リスク

SEC01-BP04

セキュリティ脅威と推奨事項を最新に保つ

AWS、業界、脅威インテリジェンスからの推奨事項を継続的に把握し、脅威モデルを進化させます。

高リスク

SEC01-BP05

セキュリティ管理範囲を削減する

管理対象のセキュリティ範囲を削減し、マネージドサービスの活用により運用負荷を軽減します。

中リスク

SEC01-BP06

標準セキュリティ制御の展開を自動化する

標準的なセキュリティ制御の展開を自動化し、一貫性と効率性を確保します。

中リスク

SEC01-BP07

脅威モデルを使用して脅威を特定し緩和策に優先順位を付ける

脅威モデルを使用して潜在的な脅威を特定し、リスクベースで緩和策の優先順位を決定します。

高リスク

SEC01-BP08

新しいセキュリティサービスと機能を定期的に評価し実装する

新しいAWSセキュリティサービスと機能を定期的に評価し、適切なものを実装します。
