SEC07-09 データ保護

データの分類、保存時および転送中のデータ保護を実装し、機密性と整合性を維持します。

SEC07

データ分類

重要度と機密性に基づいてデータを分類し、適切な保護と保持制御を決定します。

4つのベストプラクティス

SEC08

保存時のデータ保護

複数の制御を実装して、不正アクセスや誤った取り扱いのリスクを軽減し、保存時のデータを保護します。

4つのベストプラクティス

SEC09

転送中のデータ保護

複数の制御を実装して、不正アクセスや損失のリスクを軽減し、転送中のデータを保護します。

3つのベストプラクティス

SEC07: データ分類

SEC07-BP01

データ分類スキームを特定する

組織のデータに適した分類スキームを特定し、実装します。

高リスク

SEC07-BP02

データ分類に基づいてデータを分類する

定義された分類スキームに基づいてデータを適切に分類します。

高リスク

高リスク

データ保持要件を定義する

データの種類に応じた適切な保持要件を定義し、実装します。

中リスク

SEC08: 保存時のデータ保護

SEC08-BP01

安全なキー管理を実装する

暗号化キーのライフサイクル全体を通じて安全な管理を実装します。

高リスク

高リスク

保存時のデータ保護を自動化する

データ保護プロセスを自動化して一貫性と効率性を確保します。

高リスク

アクセス制御を強制する

データへのアクセスを適切に制御し、最小権限の原則を適用します。

SEC09-BP01

高リスク

高リスク
