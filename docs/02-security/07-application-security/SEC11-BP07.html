SEC11-BP07 パイプラインのセキュリティプロパティを定期的に評価する

CI/CDパイプラインのセキュリティ設定を定期的に評価し、継続的な改善を実現します。

公式ドキュメント

リスクレベル

中リスク

このベストプラクティスを活用しない場合、以下のリスクが発生する可能性があります：

  • パイプラインの脆弱性蓄積
  • セキュリティ設定の劣化
  • 新しい脅威への対応遅れ
  • コンプライアンス要件の未達
定期的な評価で
パイプラインの健全性を維持！

期待される成果

継続的に監視・評価されるパイプラインにより、セキュリティ設定の劣化を防ぎ、最新の脅威に対応できる体制を構築します。

セキュリティ向上！

実装ガイダンス

  • 定期監査：パイプライン設定の定期的なレビュー
  • セキュリティスキャン：パイプライン自体の脆弱性検査
  • アクセス制御：権限設定の定期的な見直し
  • シークレット管理：認証情報の適切な保護
  • ログ監視：パイプライン活動の継続的な監視
  • 継続的改善：評価結果に基づく改善実施
